# E-commerce Pro - Full Stack Project

Este proyecto es una plataforma de comercio electrónico desarrollada como parte de mi formación profesional en **Análisis de Sistemas**. El objetivo es demostrar mis capacidades técnicas en el desarrollo web moderno, integración de bases de datos y experiencia de usuario.

## 🚀 Tecnologías Utilizadas
* **Frontend:** React.js, CSS Modules, React Router.
* **Backend/Database:** Firebase (Firestore & Authentication).
* **Deploy:** Vercel.
