# E-commerce Pro - Full Stack Project

Este proyecto es una plataforma de comercio electrónico dinámica y funcional, desarrollada como parte de mi formación profesional en **Análisis de Sistemas**. El objetivo principal es demostrar mis capacidades técnicas en el desarrollo web moderno, la integración de servicios en la nube y la creación de experiencias de usuario fluidas.

## 🚀 Tecnologías y Herramientas
* **Frontend:** React.js (Hooks, Functional Components).
* **Routing:** React Router DOM para navegación SPA (Single Page Application).
* **Backend as a Service (BaaS):** Firebase (Firestore para base de datos y persistencia).
* **Lenguajes:** JavaScript (ES6+), HTML5, CSS3.
* **Deploy:** Vercel.

## 🌟 Funcionalidades Destacadas
* **Gestión de Catálogo Dinámico:** Consumo de datos en tiempo real desde Firestore con filtrado por categorías.
* **Sistema de Carrito:** Lógica avanzada para agregar, eliminar y calcular totales de productos.
* **Flujo de Checkout:** Proceso de finalización de compra con generación de órdenes en la base de datos.
* **Diseño Responsivo:** Interfaz optimizada para una navegación fluida en dispositivos móviles y escritorio.
